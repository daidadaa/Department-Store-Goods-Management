Help and Common Question

Q1: Why data disappear after browser close?

This project only stores data in memory of current session.

Q2: Why duplicate goods ID is blocked?

Each goods must have unique ID, then update or delete will not be wrong.

Q3: How to update one goods?

Click Edit in table, change data fields, then click Update by Goods Name.

Troubleshooting

  • Please fill required fields first
  • Quantity and price must be 0 or bigger
  • Use correct goods name for update or delete
  • If no search result, please reset filter condition
