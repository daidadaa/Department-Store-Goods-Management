Privacy and Security Analysis

This app has no backend, but basic privacy and security control still is needed.

Privacy

  • Only goods data fields are collected
  • No customer or payment information is stored
  • Data only save in current session

Security

  • Required field and number type validation
  • Unique goods ID check
  • Delete need confirmation dialog

Usability

  • Responsive layout for small screen
  • Clear feedback message
  • Simple page navigation
