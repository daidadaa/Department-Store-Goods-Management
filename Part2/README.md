# Part 2 Angular

This folder contains the Angular version of my PROG2005 inventory management assignment.

## Main Pages
- Home
- Manage Items
- Search and Filter
- Privacy and Security
- Help

## Features
- Add new item with validation
- Update item by name
- Delete item by name with confirmation
- Search by name
- Filter by category and popular status
- Display all items and popular items
- Show total items, total value, and low/out stock count

## Data Storage
This app does not use backend/database.
Records are stored in memory for current browser session only.

## Run
`npm install`
`npm start`

Open: `http://localhost:4200/`
