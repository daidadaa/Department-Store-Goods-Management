# Part 1 TypeScript

Browser-based inventory system using TypeScript.

## Features
- Add item
- Update by item name
- Delete by item name with confirmation
- Search by name
- Show all / popular only
- Select and batch delete
- Basic validation and responsive layout

## Data
Session-only memory data. Refresh or close browser will reset data.

## Run
Compile TypeScript to `dist/inventory.js` then open `index.html`.
